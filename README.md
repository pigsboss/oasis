# MEAYIN-_OASIS_
**O**bservables **A**nalysis and **S**imulation for **I**nterferometer-_in_-**S**pace

空间干涉仪观测数据分析与模拟系统

---

## 功能定位

OASIS 是一套面向科学用户的空间干涉仪软件工具，旨在帮助科学用户筹划观测任务、分析观测数据。
此外，在工程方案设计与研制阶段，OASIS 也能帮助科学用户评估方案科学目标可实现性，从而为方案设计及试验验证提供参考。

## 系统框架

OASIS 总体分为三层，分别是目标层、物理层与数据层。

 - 目标层：以观测目标（source）参数化模型为核心，产生目标的图像、光变曲线及光谱模型，或从重建得到的目标图像、光变曲线及光谱反演目标参数；
 - 物理层：以观测变量（observable）为核心，例如复可见度（visibility），产生模拟干涉采样，或从干涉采样重建观测变量；
 - 数据层：以观测数据（data）为核心，例如条纹数据，模拟产生包含各类噪声的探测器读出，或处理观测数据、复原为干涉采样。
