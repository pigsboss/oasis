# 目标层（Targets Layer）设计开发规范

## 1. 核心设计原则

### 1.1 可插拔、开放式设计

目标层采用**完全开放式架构**，支持多科学主题并行发展，确保系统的可扩展性和灵活性：

- **模块化设计**：每个科学主题作为独立模块，可独立开发、测试和部署
- **插件式注册**：新模型通过注册机制自动集成，无需修改核心代码
- **标准接口**：所有模型遵循统一的接口规范，确保互操作性
- **动态发现**：系统运行时自动发现和加载可用模型
- **版本管理**：支持模型版本控制和向后兼容

### 1.2 统一输出接口

无论何种目标类型、何种物理模型，都必须能够产生以下三种标准输出，确保与观测量层的无缝对接：

- **图像模型**：二维亮度分布（可包含时间/波长维度）
- **光谱模型**：波长-通量关系（可包含时间维度）
- **光变模型**：时间-通量关系（可包含波长维度）
